# hey

curiosity-driven founder. i design, build, and ship products.

i take ideas from nothing to production - design, code, copy, infrastructure, the whole thing.

right now i'm building infrastructure for AI agents: orchestration, memory, testing, security, observability. agents that can browse, email, file taxes, and ship code need real infrastructure underneath. that's what i'm working on.

## agent infrastructure

- [loop](https://github.com/acoyfellow/loop) - orchestrate AI coding loops with guardrails
- [deja](https://github.com/acoyfellow/deja) - persistent memory for agents. semantic recall across runs
- [gateproof](https://github.com/acoyfellow/gateproof) - e2e testing harness: observe logs, run actions, assert results
- [chomp](https://github.com/acoyfellow/chomp) - openai-compatible proxy across 6 free/cheap model providers
- [benchmarks](https://github.com/acoyfellow/benchmarks) - tool-calling reliability & latency tracking for CF models
- [effect-first](https://github.com/acoyfellow/effect-first) - token-aware reference API for agents writing Effect programs
- [homerun](https://github.com/acoyfellow/homerun) - local HTTP proxy so cloud agents browse through your residential IP
